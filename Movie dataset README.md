# Movie Industry Analysis Project

## Project Overview:
This project explores a movie industry dataset using Python to understand relationships between different variables in the dataset such as budget, gross earnings, votes and ratings. 

The key object is to identify the factors that are most strongly correlated with gross revenue for movies. 

---

## Tools:
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## Skills:
- Data loading  
- Data cleaning and missing values  
- Conversion of data types
- Removing duplicates   
- Data visualisation with heatmaps and scatter plots  
- Analysing correlation 

---

## Key Insights & Findings

- Budget and gross revenue had a strong positive correlation, suggesting that movies with higher budget generate higher earnings.

- Votes and gross revenue had a strong positive correlation, suggesting that movies with more reviews generate higher earnings. 

- Movie ratings had a weaker correlation with gross revenue, suggesting that this is not a key factor for finacial performance. 

- Other factors such as genre and company also had weak correlations with gross earnings so these are not key factors for gross earnings.
